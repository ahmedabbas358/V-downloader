## 🌟 What is V-Downloader?

V-Downloader is an advanced media downloader app for Android that allows you to download videos and audio from thousands of websites. Built upon the robust foundation of `yt-dlp` and `Aria2c`, this version introduces a sleek new Apple-style Glassmorphism UI, powerful batch downloading capabilities, and absolute privacy.

## ✨ Key Features & Enhancements

- **🍏 Apple Glassmorphism Design:** A completely revamped, modern UI featuring frosted glass effects, smooth squircle shapes, and elegant dark mode aesthetics that feel premium and native.
- **🚀 Ultra-Fast Concurrent Downloads:** Full control over your background download speed! Utilize the new settings slider to download up to **10 playlists or items simultaneously** in the background.
- **🌍 Full Arabic Support (RTL):** 100% localized in Arabic, ensuring a seamless right-to-left experience without any visual glitches.
- **🛡️ Maximum Privacy (No Tracking):** V-Downloader respects your privacy. All background telemetry and auto-updater phone-home features have been completely stripped from the code. Your phone only talks directly to the sites you are downloading from.
- **🎥 Supports 1,000+ Sites:** Download from virtually anywhere using the power of `yt-dlp`.
- **🎵 Flexible Formats:** Extract audio, save thumbnails, embed subtitles, and choose your preferred video quality with ease.

## 📥 How to Download & Install

Since you are viewing this repository, you can easily download the latest `APK` generated by GitHub Actions:

1. Go to the **Actions** tab in this repository.
2. Click on the latest workflow run (e.g., "Build Release APK").
3. Scroll down to the **Artifacts** section at the bottom of the page.
4. Download the `debug-apks` zip file.
5. Extract it and install the `V-Downloader-...-debug.apk` on your Android device!

## ⚙️ Building from Source

If you prefer to build it yourself, you don't even need Android Studio. You can fork this repository, and GitHub Actions will automatically compile the APK for you!
